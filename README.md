# 电商订单数据分析报告
## 项目背景
基于 189,356 条订单记录，使用 PostgreSQL 进行多维度分析，挖掘销售趋势、用户偏好和品类表现。
数据来源：https://www.kaggle.com/datasets/dongxiaol/test-orders-info
## 一、整体销售指标
总销售额 ：约 9.47 亿 
总订单数 ：189,356 
客单价 ：约 500 元 
## 二、月度销售趋势
- 各月销售额占比稳定在 **7.6% - 8.5%** 之间，2 月略低（7.62%），3 月和 12 月为小高峰。
跨月高销量产品（出现 ≥2 次）：
牛仔裤：出现在 6 月、8 月
笔记本电脑：出现在 1 月、10 月
 智能手机：出现在 2 月、3 月
> 这些产品具有持续热卖能力，建议保持库存充足并作为主推款。
## 三、支付方式交叉分析
- 四种支付方式（微信支付、支付宝、信用卡、银行卡）的 **用户性别比例均接近 1:1**，无明显差异。
- 各支付方式下 销售额 Top3 类别高度重叠：食品、家居、电子产品、美妆、服装。
- 各支付方式的 区域分布非常均匀主要城市占比均在 20% 左右），无地域聚集性。
 结论：支付方式选择与用户性别、品类偏好、地域均无强关联，无需差异化运营。
## 四、年龄 + 性别分组分析
### 4.1 销售额分布
| 年龄分组 | 销售额（亿） | 占比 |
| 18-30 | 约 2.86 | 30.2% |
| 31-40 | 约 2.21 | 23.4% |
| 41-50 | 约 2.19 | 23.1% |
| 51-60 | 约 2.21 | 23.3% |
 **核心洞察**：**18-30 岁群体贡献近三分之一销售额**，远高于其他年龄段。
### 4.2 各年龄段偏好产品类别
| 年龄分组 | 男性偏好 Top2 | 女性偏好 Top2 |
| 18-30 | 食品、服装 | 服装、家居 |
| 31-40 | 食品、家居 | 服装、电子产品 |
| 41-50 | 食品、美妆 | 服装、电子产品 |
| 51-60 | 美妆、电子产品 | 电子产品、家居 |
 **核心洞察**：
- **食品** 在 18-50 岁男性中始终排名靠前
- **美妆** 在 41 岁以上男性中偏好显著上升
- **电子产品** 在 31-60 岁女性中持续热门
## 五、核心结论与建议
1. **年轻用户是绝对主力**：18-30 岁贡献 30% 销售额，应重点投放食品、服装、美妆类广告。
2. **跨月爆品明确**：牛仔裤、笔记本电脑、智能手机为持续热销款，可绑定组合促销。
3. **中老年男性美妆市场值得挖掘**：41-60 岁男性对美妆产品已表现出明显偏好。
4. **支付方式无需区别运营**：各渠道用户画像高度一致，可统一策略。
